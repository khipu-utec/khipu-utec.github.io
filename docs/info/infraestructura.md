## Software del sistema

- **Sistema Operativo:** Rocky Linux 8.10 (Green Obsidian)
- **Message Passing Library:** MPICH
- **Compiladores:** Intel, GCC
- **Job Scheduler:** SLURM 
- **Manejo de software:** Módulos de ambiente
